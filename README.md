# Linda Fiorini personal website

Sito statico personale pronto per GitHub Pages.

## Come pubblicarlo

1. Vai su GitHub e crea una repository pubblica chiamata esattamente:

   `lindafiorini95.github.io`

2. Carica tutti questi file nella repository:
   - `index.html`
   - `assets/style.css`
   - `robots.txt`
   - `sitemap.xml`
   - `.nojekyll`

3. Apri la repository su GitHub → Settings → Pages.

4. In “Build and deployment”, scegli:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root

5. Dopo qualche minuto il sito sarà disponibile qui:

   `https://lindafiorini95.github.io/`

## Cose da personalizzare

- Sostituisci eventuali link non corretti o mancanti.
- Aggiungi una foto professionale, se vuoi.
- Aggiungi un indirizzo email professionale solo se vuoi renderlo pubblico.
- Dopo la pubblicazione, usa Google Search Console per richiedere l'indicizzazione.
